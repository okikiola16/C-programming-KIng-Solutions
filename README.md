# C programming: A Modern Approach - My Solutions

My solutions to exercises from K.N. King's "C Programming: A Modern Approach" (2nd Edition).

## Structure
- `project1/` - Early chapter exercises
- `project2/` - Later chapter exercises

## Exercises Included

### Project 1
- 2dweight: 2-dimensional weight calculation
- addfrac: Fraction addition
- celsius: Celsius to Fahrenheit conversion
- dweight: Dimensional weight calculation
- greedy: Greedy algorithm change calculation
- Horners-rule: Polynomial evaluation using Horner's Rule
- interest: Interest calculation
- mod-sphere: Modified sphere volume calculation
- parkinson: Parkinson's Law calculation
- polynominal: Polynomial evaluation
- pun: My first C program
- sphere: Sphere volume calculation
- tick: Stock ticker simulation
- tprintf: Formatted printing exercise
- VAT: Value Added Tax calculation

### Project 2
- cashier: Cash register program
- date: Date formatting and manipulation
- display: Number display program
- isbn: ISBN number parser
- phone-number: Phone number formatter
